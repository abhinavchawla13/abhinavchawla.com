## Project setup
#### Install dependencies
Make sure you have Node.js (v12+, npm v6+) installed
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```

#### Lints and fixes files
```
npm run lint
```
